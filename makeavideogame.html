Variables!

What's a Variable?


Variables are an essential part of programming! They help the computer remember everything that we want to use. For example, let’s say I want the computer to remember my name. I can use a variable so that the computer remembers what my name is. Here’s how we can write it in python:


  • my_name = “Jacob”

Here, I've called the variable “my_name” and the value that it's pointing to is “Jacob”. This means that the computer now knows that “my_name” is pointing to “Jacob”.

Basic Data Types

×

As we have seen, we can store different “things” in variables. These “things” are called Data Types. Here are the most common data types:


  1. String: Strings are groups of characters, like “Hello world”. Strings are surrounded by quotes (“”). If you remember back to when we printed “Hello world”, we wrapped it in quotes. If we hadn’t, the program wouldn’t have compiled; or, in more basic terms, it wouldn’t have worked. We’ll see why in a couple of lessons.
  2. Integers: Integers - or ints for short - are very similar to their math counterparts. An int is simply a whole number that can be positive or negative, such as 1, 2, -5, 100.
  3. Floats: Floats - or floating point numbers - are all the other numbers that ints don't cover. Floats are similar to real numbers in math, such as 3.1415, -2.71, 0.567, 3.0, etc.
  4. Boolean: Boolean - or bools - contain only two values: True or False. If you've ever played the game 21 questions, you can imagine boolean values as similar to "Yes" or "No" answers.

Conditionals

×

Conditionals are a way to make decisions. They tell the computer to do something if a certain condition is met (hence, the name conditionals). For example, if you want to print “Hello” only if the user’s name is "John", you would use a conditional. If the condition, in this case the user's name being "John", is met, then the computer will print “Hello”. If the condition is not met, then the computer will not print “Hello”. Here’s how we can write this in python:


if CONDITION:

Do something

else:

Do something else


Notice that a condition will always evalute to either True or False. This means if this condition is true, do this. If it's false, do this instead. Thus, comparing values can be very helpful. Read this article on how we can compare values. Also watch this video if you're still confused.
