What's a Variable?


Variables are an essential part of programming! They help the computer remember everything that we want to use. For example, let’s say I want the computer to remember my name. I can use a variable so that the computer remembers what my name is. Here’s how I can write it in python:

  • my_name = “Jacob”

The variable name is “my_name” and the value is “Jacob”. This means that the computer now knows that “my_name” is pointing to “Jacob”.
