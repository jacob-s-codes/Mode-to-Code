How to Start

Begin by selecting a topic that you're interested in, such as making a website or a videogame! From there, follow the curated cirriculum that I have created which maximizes the free resources accross the internet.

What is Mode to Code?

Learning to code can feel overwhelming, especially considering the vast amount of resources on the internet. Mode to Code provides roadmaps which take the thousands of resources on the internet and provide clear, step-by-step instructions for learning how to code. These roadmaps are intended for middle and high school students who are interested in learning about computer science. Start by choosing a path and follow it to create websites, games, and more!
